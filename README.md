An Add-On Mod for Balatro that provides the ability to view more jokers at once in the collection, as well as an Auto-Reroll feature that allows the player to search for multiple cards at a time in the shop.


Controls:
- Right Click the reroll button to open the collection
- Shift + Right Click to start rerolling (will open collection if no cards are currently being searched for)
- Shift + Double Click to clear the list of cards to search
